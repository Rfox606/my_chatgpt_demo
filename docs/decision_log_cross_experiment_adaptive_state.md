# 决策记录：跨实验自适应状态 v1

- **2026-07-21 — 固化目标。** 本版本从“试验内的连续状态/预警”转向“历史实验先验 + 目标前缀受约束适配”的跨实验进程监测。
- **范围约束。** 共同语义是相对退化进程位置与局部活跃度，绝非绝对磨损量；`progression_score` 与 `activity_score` 必须分离。
- **边界。** Stage、形貌、磨屑和未来目标数据被排除在正式模型边界之外；它们只允许完整推理后的诊断使用。
- **适配策略。** 源模型冻结；仅受限残差适配器可在线更新。前 1000 有效周期不更新，且只读取到达的目标前缀。
- **不可回退。** 实现和结果无论成功或失败均不改变本文档的目标、非目标、术语或接受标准；不得改回五阶段分类、绝对磨损比较、纯目标自基线偏离或 v4.5 四状态预警。
