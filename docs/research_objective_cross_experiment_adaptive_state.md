# 跨实验自适应退化进程监测：研究目标与边界（ceap_v1）

## 最终研究目标

从历史磨损实验中学习**可迁移的退化进程状态表征**，使新实验在接入时获得来自历史实验的非零状态先验。随着目标实验力信号的前缀逐步到达，仅对有限的实验特异性残差参数进行因果、受约束的在线更新，以逐步提高该实验退化进程判断的准确性与可信度。

## “跨实验”的准确含义

- 历史实验知识可迁移到新实验；各实验共享状态解释框架，而不是共享绝对物理量。
- 目标实验不从自身早期基线重新归零；其初始 `progression_score` 保留源实验提供的先验。
- 到达的目标前缀只可校正尺度、偏置、演化速度与路径权重；源模型主体始终冻结。
- Exp1 与 Exp2 允许有不同的状态路径：一个实验可处于较后、相对稳定的进程位置，另一个可处于较后且持续活跃的位置。模型必须把进程位置和动态活跃度分开。

## 正式输出及术语边界

- `progression_score`：实验条件化的退化进程位置分数，不是绝对磨损量。
- `activity_score`：当前状态变化活跃度，用于区分“后期稳定”和“后期持续活跃”，不是绝对磨损量。
- `state_uncertainty`：当前进程判断的不确定度，由特征配置、源域支持/OOD、目标证据量、先验—适配分歧、适配器边界和局部波动共同构成。

这些输出不得解释为绝对磨损百分比、剩余寿命、临床风险概率、跨材料相同损失，或跨实验可直接比较的磨损质量/体积/深度/碎屑数量。

## 数据和因果边界

正式训练和在线适配的主输入为 `outputs_continuous_state_v45/results/window_feature_raw_v45.csv` 中预先声明的力信号特征。第一版标准特征为：`rx_mean`、`rx_absmean`、`rx_q05`、`ry_mean`、`ry_absmean`、`ry_q05`、`ry_p2p`、`rs_mean`、`rs_rms`。

以下字段绝不进入正式模型训练或在线适配：`Stage`、形貌指标（Sa/Sq/Sz/Sku 等）、磨屑计数、未来目标窗口，以及 v4.5 混合定义的共识分数。Stage 和形貌只能在完整推理后用于事后诊断。cycle 不作为模型输入，只可作为构造源域时间有序对、前缀审计和评估索引。

源域模型用有 embargo 的源域时间训练/验证和 pairwise ranking 训练；超参仅在源域验证段选择，随后由完整源域重训。在线阶段只使用已经到达的目标窗口/目标时间对；前 1000 有效周期冻结，残差从零开始，`lambda_t` 渐进且有上限，更新受 L2、参数范数和单次步幅约束。高波动、短时突变和严重 OOD 必须暂停或降权更新。

## v4.5 的定位

v4.5 的 `D_state`、`V1000`、多尺度差异和波动性仅可作为目标域局部动态、适配门控和不确定度证据；不得包装为最终跨实验健康/退化分数，也不得把本工作改写为 v4.5 四状态预警。

## 明确非目标与禁止偏移

不得：

1. 改回跨实验五阶段分类，或把 Stage 作为在线适配标签；
2. 比较 Exp1/Exp2 的绝对磨损质量、体积、深度或总磨屑数，或将同分数解释为相同材料损失；
3. 将目标实验自身早期基线偏离定义为最终进程分数，或强制接入起点为零；
4. 强制所有实验形成同一条轨迹、相同阶段边界、完整轨迹或终止语义；
5. 自由更新共享源模型，或以形貌选择模型、特征或阈值；
6. 把 cycle 或“接入后经过时间”作为正式模型输入，替代跨实验状态迁移；
7. 只报告成功方向，或在运行后修改接受标准。
